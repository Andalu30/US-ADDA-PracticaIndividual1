La clase {@link us.lsi.iterativo.Matrices Matrices} tiene métodos para construir las matrices de fracciones de BigInteger, necesarias para resolver problemas de secuencias: la base, la matriz unidad y el vector columna con los valores iniciales. De manera similar podríamos construir matrices de otros campos como los números complejos o los números reales con diferentes implementaciones.
