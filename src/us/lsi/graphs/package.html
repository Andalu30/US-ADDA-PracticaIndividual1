Un conjunto de ejercicios de grafos pueden encontrarse en: ProblemasDeGrafos

Las clases usadas en el paquete son:

  • La clase {@link us.lsi.graphs.SimpleVirtualGraph SimpleVirtualGraph} implementa un grafo virtual simple
  • La clase {@link us.lsi.graphs.UndirectedSimpleVirtualGraph UndirectedSimpleVirtualGraph} implementa un grafo virtual simple y no dirigido.
  • La clase {@link us.lsi.graphs.DirectedSimpleVirtualGraph DirectedSimpleVirtualGraph} implementa un grafo virtual simple y dirigido.
  • La clase {@link us.lsi.graphs.VirtualVertex VirtualVertex} implementa un vértice de un grafo virtual simple
  • La clase {@link us.lsi.graphs.ActionVirtualVertex ActionVirtualVertex} implementa un subtipo de VirtualVertex adecuado como vértice de un grafo virtual simple cuyas aristas están definidas por un conjunto finito de acciones codificadas en una lista
  • La clase {@link us.lsi.graphs.VirtualDirectedVertex VirtualDirectedVertex} implementa un vértice de un grafo virtual simple y dirigido
  • La clase {@link us.lsi.graphs.SimpleEdge SimpleEdge} implementa una arista entre dos vértices
  • La clase {@link us.lsi.graphs.GraphPaths GraphPaths} implementa una factoría del tipo GraphPath
  • La clase {@link us.lsi.graphs.GraphsFileExporter GraphsFileExporter} es adecuada para grabar grafos en un fichero
  • La clase {@link us.lsi.graphs.GraphsReader GraphsReader} es adecuada para leer grafos de un fichero
  • La clase {@link us.lsi.graphs.GraphView GraphView} implementa una vitsa de un grafo donde los vértices se identifican con enteros
