Implementaciones de distintos tipos de cromosomas.

Las tipos de cromosomas definidos definidos son:

  • El tipo {@link us.lsi.ag.agchromosomes.IChromosome IChromosome} representa un cromomosoma en general.
  • La clase {@link us.lsi.ag.agchromosomes.IBinaryChromosome IBinaryChromosome} representa un cromosoma cuyo valor decodificado es una lista de enteros del tamaño especificado en el problema.
  • El tipo {@link us.lsi.ag.agchromosomes.IndexChromosome IndexChromosome} representa un cromomosoma adecuado para resolver problemas cuya solución es un Multiset o una lista, posiblemente con elementoss repetidos, formados con elementos de un conjunto dado
  • El tipo {@link us.lsi.ag.agchromosomes.IExpressionChromosome IExpressionChromosome} representa un cromosoma cuyo valor decodificado es una expresión formada por un conjunto de operadores dados, un conjunto de variables y un conjunto de constantes.
  • El tipo {@link us.lsi.ag.agchromosomes.IRealChromosome IRealChromosome} representa un cromomosoma cuyo valor decodificado es un lista de número reales

Implementaciones de distintos tipos de cromosomas.

Las clases usadas en el paquete son:

  • La clase {@link us.lsi.ag.agchromosomes.BinaryChromosome BinaryChromosome} implementa un cromosoma binario. Es decir uno cuya valor decodificado es una lista de ceros y unos. La implementación es una adaptación de la clase de Apache BinaryChromosome
  • La clase {@link us.lsi.ag.agchromosomes.IndexSubListChromosome IndexSubListChromosome} implementa el tipo {@link us.lsi.ag.agchromosomes.IndexChromosome IndexChromosome}. El valor decodificado de este cromosoma es un subconjunto de la secuencia normal definida en el problema. La implementación reutiliza la clase de Apache BinaryChromosome
  • La clase {@link us.lsi.ag.agchromosomes.IndexRangeChromosome IndexRangeChromosome} implementa el tipo {@link us.lsi.ag.agchromosomes.IndexChromosome IndexChromosome}. El valor decodificado de este cromosoma está formada por una lista de tamaño n cuyos elementos para cada i están en el rango rango 0..m(i), siendo m(i) la multiplicidad máxima para i definida en el problema.
  • La clase {@link us.lsi.ag.agchromosomes.IndexPermutationChromosome IndexPermutationChromosome} implementa el tipo {@link us.lsi.ag.agchromosomes.IndexChromosome IndexChromosome}. El valor decodificado de este cromosoma es una permutación de la secuencia normal definida en el problema. La implementación reutiliza mediante herencia la clase de Apache RandomKey
  • La clase {@link us.lsi.ag.agchromosomes.IndexPermutationSubListChromosome IndexPermutationSubListChromosome} implementa el tipo {@link us.lsi.ag.agchromosomes.IndexChromosome IndexChromosome}. El valor decodificado de este cromosoma es un subconjunto de una permutación de la secuencia normal definida en el problema. La implementación reutiliza mediante composición las clases de Apache RandomKey y BinaryChromosome
  • La clase {@link us.lsi.ag.agchromosomes.ExpressionChromosome ExpressionChromosome} implementa el tipo {@link us.lsi.ag.agchromosomes.IExpressionChromosome IExpressionChromosome}. La implementación sigue las ideas de Gene expression programming
  • La clase {@link us.lsi.ag.agchromosomes.ChromosomeFactory ChromosomeFactory} implementa una factoría de cromosomas de los tipos anteriores

Otras clases del paquete son:

  • La clase {@link us.lsi.ag.agchromosomes.ListIntegerChromosome ListIntegerChromosome} implementa un cromosoma cuyo valor decodificado es una lista de enteros del tamaño especificado en el problema. Según los operadores de mutación y cruce usados obtendremos unas listas de enteros determinadas.
