Una explicación de la Programación Dinámica puede encontrarse en Programación Dinámica

La documentación puede encontarse en el: Tema13.

Los tipos definidos en el paquete son:

  • la clase {@link us.lsi.pd.AlgoritmoPD AlgoritmoPD} es el algoritmo para resolver los problemas de Programación Dinámica.
  • La clase {@link us.lsi.pd.AlgunosTestPD AlgunosTestPD} contiene tests para los problemas de PD.
  • El interface {@link us.lsi.pd.ProblemaPD ProblemaPD} tiene que ser implementado por un problema para poder ser resuelto por el algoritmo {@link us.lsi.pd.AlgoritmoPD AlgoritmoPD}
  • El tipo {@link us.lsi.pd.ProblemaPDR ProblemaPDR} es una adaptación de {@link us.lsi.pd.ProblemaPD ProblemaPD} para el caso de reducción.
